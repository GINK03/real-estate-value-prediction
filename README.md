# Real Estate Value Prediction
Awesome real estate value prediction with machine-learnings.

## 序
 - 機械学習で不動産を予想する意味  
  特徴量から重要度を知ることができる
  
 - EndUserにとって嬉しいこととは？  
  Chrome Extentionなどでその情報の正当性を推し量る事ができる
  
## 破
 - データを集める  
 Link
 
 - モデルを検討する
 独立な情報でone-hotベクトルを仮定  
 Elastic-Netを利用  
 optunaで最適化
 
 - 東京と全国でモデルを分ける
 
 - Chrome Extentionを作る
